このページについて

火加減手帖（以下「当サイト」）は、当サイトを利用されるすべての方のプライバシーを尊重します。本ポリシーは、当サイトにおける個人情報の取り扱い、Cookie・アクセス解析、アフィリエイトプログラムの利用について定めるものです。

1. 取得する情報と取得方法

当サイトは、お問い合わせの際にご入力いただく情報を取得します。お問い合わせはGoogleフォームを通じて受け付けており、お名前（またはニックネーム）、メールアドレス、お問い合わせ内容などをご入力いただく場合があります。これらの情報は、お客様が自発的にご提供いただいた範囲でのみ取得します。

当サイトを閲覧するだけの場合、氏名・住所・電話番号などの個人情報を入力していただく必要はありません。

2. 利用目的

取得した情報は、以下の目的のためにのみ利用します。

  • お問い合わせへの返信・対応のため
  • 当サイトの運営・改善に役立てるため

利用目的の範囲を超えて個人情報を利用することはありません。

3. アクセス解析（Cloudflare Web Analytics）

当サイトは、サイトの利用状況を把握するために、インフラ提供事業者であるCloudflare, Inc. が提供する「Cloudflare Web Analytics」を利用しています。このアクセス解析はCookieを使用せず、訪問者個人を識別したり、サイトをまたいで行動を追跡したりしません。収集されるのはページの表示回数や参照元などの匿名の統計情報のみです。

また、当サイトはCloudflareのCDN・セキュリティ機能を通じて配信されており、その過程でアクセス元のIPアドレス等が、サイトの安定運用・セキュリティ確保の目的でCloudflareにより処理される場合があります。詳細はCloudflareのプライバシーポリシーをご確認ください。

4. Cookie・アフィリエイトプログラムについて

当サイトは、A8.net・もしもアフィリエイトをはじめとするアフィリエイトプログラムに参加しています。これらのプログラムでは、広告主やアフィリエイト・サービス・プロバイダ（ASP）などの第三者が、成果計測（どの広告から申し込みがあったかの判定）のためにCookieを使用することがあります。

これらのCookieによって個人を特定できる情報が当サイトに送信されることはありません。Cookieの送受信は、お使いのブラウザの設定により無効にすることができます。無効化の方法はご利用のブラウザのヘルプをご参照ください（無効にした場合、一部のサービスが正しく動作しないことがあります）。

5. 第三者への提供

当サイトは、法令に基づく場合を除き、ご本人の同意なく個人情報を第三者へ提供することはありません。

6. 個人情報の開示・訂正・削除

ご本人から、当サイトが保有する個人情報の開示・訂正・削除のご請求があった場合は、ご本人であることを確認のうえ、合理的な範囲で速やかに対応します。ご請求はお問い合わせよりご連絡ください。

7. 免責事項

当サイトに掲載する情報の正確性には努めていますが、その内容を保証するものではありません。当サイトの利用により生じた損害について、当サイトは責任を負いかねます。詳細は免責事項をご確認ください。

8. プライバシーポリシーの変更

当サイトは、法令の変更やサービス内容の変更に応じて、本ポリシーを予告なく変更することがあります。変更後の内容は、本ページに掲載した時点から効力を生じるものとします。
